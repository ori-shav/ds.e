# A practical deep dive into building, scaling and maintaining high-quality design systems for software engineers course code.

This is a monorepo that uses `yarn` for package management.

## Requierments

- `npm` `8.1.2`
- `node` `16.13.1`
- `yarn` `1.22.17`

## Useage

```
$ yarn
$ cd packages/react
$ yarn start-storybook
```
